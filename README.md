# Slotify

Modern appointment booking SaaS platform for businesses.

## Current Status

⚠️ This project is currently under active development and is not production ready yet.

Core booking flow and availability management are being built and improved continuously.

## Features In Progress

- Business booking pages
- Dynamic appointment slots
- Weekly availability management
- Break time management
- Appointment confirmations
- Responsive booking UI
- Admin dashboard
- Real-time availability logic

## Tech Stack

- Laravel
- Vue 3
- TypeScript
- Inertia.js
- MySQL

## Notes

This repository is a work in progress and the architecture, UI, and features may change frequently during development.
